通过输入组件类型名来添加组件，功能类似编辑器3.0的添加组件。

## 功能介绍
- 2.X存在一些BUG，有时不能通过拖拽的方式增加组件到节点中，那就只有鼠标点击的方式。当自定义组件很多的时候，列表会很长。
- 这个组件让你像3.X那样，输入关键字来查找并添加组件。
- 注意：面板必须挂靠在creator界面中才能正确读取到用户自定义组件。

## 使用教程
- 选择节点（可以选择多个）
- 输入类名，回车。（或点击下方列表中某个项）
